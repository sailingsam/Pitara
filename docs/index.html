GitHub backs up your code.
Pitara backs up your environment.

Set up a new laptop in minutes instead of a lost day. Pitara captures your language runtimes and global CLI tools and reinstalls them on a new machine.

curl -fsSL https://pitara.dev/install.sh | sh
macOS & Linux — no Node.js needed, auto-detects your system
★ Star on GitHub View on npm

Trust model

🔒 You own it. We never see it.

No server, no database — backups live in your own GitHub repo

No secrets stored — snapshots hold only tool names and versions

Your token never leaves your machine (~/.pitara)

100% Open Source

Audit every line yourself — no black boxes

Workflow

Three commands. That's the whole flow.

$ pitara login
Authenticate — One-time GitHub sign-in. No password, no secret stored on any server.
$ pitara backup
Snapshot — Saves to a private 'pitara-snapshots' repo in your own GitHub account.
$ pitara restore
Rebuild — Reinstalls everything on a new machine in dependency order.

Coverage

What it captures

Language runtimes Package managers Global CLI tools many more →

Installed a tool with npm? Pitara reinstalls it with npm. It remembers which package manager you used — and never mixes them up.

Don't see a tool you use? It's not a hard limit — request it or add it yourself. Every tool is just one small plugin.

Contribute

Open source.

Pitara is MIT-licensed and built around a small plugin interface. The best first contribution is a new tool: implement the Plugin interface, register it, open a PR. Issues and ideas welcome too.
